SIDEVIEW

This is the experimental code for the SIDEVIEW project - exploring partial perspective corrections in the browser. This is intended for experimental purposes.

By Frode Eika Sandnes, Oslo Metropolitan University, April, 2022.

The project comprises three prototypes

  1. Simple manual perspective correction (2D transformation, scale and shear)
  2. Manual pseudo perspective correction (3D transformations, perspective and X,Y, Z-rotation)
  3. Automatic perspective correction using camera (2D scale and shear)
